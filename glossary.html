在留・ビザ
たびレジ たびレジ

外務省の海外旅行者・在留者向け登録システム。安全情報をメールで受信できる。

外務省が運営する「外務省海外旅行登録システム」の通称。海外在留・渡航の際に登録することで、緊急情報・安全情報をメールで受け取れる。短期渡航者も登録可能。在留邦人は「在留届」の提出とあわせてたびレジへの登録が推奨される。緊急時の安否確認にも活用される。

Iqama إقامة イカーマ

サウジアラビアの外国人居留許可証。就労・生活に必須。

サウジアラビアで就労・居住する外国人に発行される居留許可証（在留カード）。雇用主がスポンサー（カフィール）となり申請する。有効期間は通常1年または2年で、更新が必要。各種行政手続き・銀行口座開設・SIM購入などすべての場面で必要となる。

Exit Re-Entry Visa تأشيرة الخروج والعودة イグジット・リエントリー・ビザ

サウジを一時出国し再入国するために必要な許可。iqama保持者が取得。

iqama（居留許可）保持者がサウジを出国して再入国する際に必要なビザ許可。Absherアプリから申請可能。有効期間は通常6ヶ月または1年。家族帯同の場合は配偶者・子どもの分も別途申請が必要。期限切れのまま出国すると再入国に支障が出るため注意が必要。

関連: iqama / absher / kafeel

Kafeel / Kafala كفيل / كفالة カフィール / カファーラ

外国人労働者のスポンサーとなる雇用主・保証人制度。

カファーラ制度は外国人労働者のスポンサーシップ制度。雇用主（カフィール）が労働者のビザ・iqama・出国許可を管理する。近年の改革（Vision 2030）により、一定条件下で雇用主の承認なしに転職できる制度が導入されつつある。

宗教・文化
Adhan（アザーン） الأذان アザーン

礼拝の時刻を告げるイスラームの呼びかけ。モスクのスピーカーから流れる。

礼拝（サラート）の時刻に、ムアッジン（礼拝呼びかけ人）がモスクから発する呼びかけ。「アッラーフ・アクバル（神は偉大なり）」の言葉で始まる。サウジではモスクのスピーカーから周辺に大音量で流れるため、在留外国人もすぐに慣れる。1日5回、礼拝の数分〜十数分前に流れる。

関連: fajr / dhuhr / asr / maghrib / isha
Asr（アスル） العصر アスル

午後中頃の礼拝。

1日5回の礼拝のうち3番目。太陽が西に傾いた午後中頃（概ね午後3〜4時台）に行われる。

関連: fajr / dhuhr / maghrib / isha
Isha（イシャー） العشاء イシャー

夜の礼拝。1日5回の礼拝のうち最後のもの。

1日5回の礼拝のうち5番目で最後のもの。夜（概ね日没後1.5〜2時間）に行われる。ラマダーン中は夜の活動が活発になるため、イシャー後も多くの商店が深夜まで営業することが多い。

関連: fajr / dhuhr / asr / maghrib / adhan
Juma（ジュムア） الجمعة ジュムア

金曜日。イスラームの集団礼拝の日。多くの企業・官公庁が休日。

アラビア語で「金曜日」を意味し、イスラームにおいて特別な集団礼拝（サラートゥル・ジュムア）が行われる聖なる曜日。サウジアラビアの週末は金曜・土曜。官公庁・多くの企業はこの曜日が休日。外国人も金曜の礼拝時間帯（正午頃）はショッピングモールの一時閉店など影響を受ける。

関連: adhan / fajr
Dhuhr（ドゥフル） الظهر ドゥフル

正午（太陽が頂点を過ぎた直後）の礼拝。

1日5回の礼拝のうち2番目。太陽が天頂を過ぎた直後（概ね正午〜午後1時）に行われる。ビジネスアワーに重なるため、多くのショッピングモール・レストランが一時的に短縮営業または閉店する。

関連: fajr / asr / maghrib / isha
Haram（ハラーム） حرام ハラーム

イスラーム法で禁じられたもの。豚肉・アルコールなどが代表例。

アラビア語で「禁じられたもの・違法なもの」を意味する。食品では豚肉・アルコールが代表的。行為としては賭博・婚前交渉・偶像崇拝なども含まれる。在留外国人はサウジの法律の下でこれらの禁止事項に従う義務があり、違反した場合は法的制裁の対象となることがある。「ハラーム」と「聖域（マッカ・マディーナ）」は同じ語根だが異なる文脈で使用される。

関連: halal
Halal（ハラール） حلال ハラール

イスラーム法で許されたもの。食品・飲料・行為などに使われる。

アラビア語で「許されたもの・合法なもの」を意味する。食品では豚肉・アルコールを含まず、イスラームの規定に従って屠畜された肉（ザビーハ）であることが条件。サウジではほぼ全ての食品がハラール対応だが、輸入食品の成分表示確認は重要。アルコール飲料はサウジ国内で法律により販売・消費が禁止されている。

関連: haram
Fajr（ファジュル） الفجر ファジュル

夜明け前の礼拝。1日5回の礼拝のうち最初のもの。

イスラームの1日5回の義務礼拝（サラート）のうち最初のもの。夜明け（サブフ）前の薄明かりの時間帯に行われる。時刻は日の出前約1〜1.5時間で、季節・緯度によって変動する。ラマダーン月中は特に重要な礼拝の一つ。

関連: dhuhr / asr / maghrib / isha / adhan
Maghrib（マグリブ） المغرب マグリブ

日没直後の礼拝。ラマダーン中はイフタール（断食明け）の合図。

1日5回の礼拝のうち4番目。日没直後に行われる。ラマダーン（断食月）中は、マグリブのアザーン（礼拝の呼びかけ）がイフタール（iftār / إفطار：断食明けの食事）の開始合図となる。この時間帯は交通渋滞が激増するため注意が必要。

関連: fajr / dhuhr / asr / isha / adhan / iftar

Nitaqat / Saudization نطاقات / سعودة ニタカット / サウダイゼーション

企業にサウジ人雇用の一定比率を義務付ける制度。

HRSDが運営する企業別サウジ人雇用義務率（Saudization）の管理制度。企業は雇用するサウジ人の比率によってPlatinum・Green・Yellow・Redのゾーンに分類され、ゾーンによって新規iqama申請や政府サービスへのアクセスが制限される。Vision 2030の目標達成に向け、業種別の比率が継続的に改定されている。

SAR ريال سعودي エスエーアール / サウジリヤル

サウジアラビア・リヤル。通貨単位。1SAR≒40〜42円（変動）。

Saudi Arabian Riyal（サウジアラビア・リヤル）の略称。通貨記号はSR または ﷼。米ドルに対して長年固定レート（1USD = 3.75SAR）を維持しているため、円ドルレートに連動して円換算額が変動する。補助通貨はハララー（1SAR = 100ハララー）。

VAT ضريبة القيمة المضافة ブイエーティー / 付加価値税

付加価値税。サウジでは2020年より15%。ほぼ全ての購入に課税。

Value Added Tax（付加価値税）。サウジアラビアは2018年に5%で導入し、2020年7月から15%に引き上げた。食料品・医薬品・教育・金融サービスなど一部は非課税または低税率の例外あり。レシートには「VAT番号」と税額が明記されており、法人は四半期ごとにZATCAへ申告義務がある。
